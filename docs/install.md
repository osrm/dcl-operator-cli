### Building from source
- clone the repository 
```
git clone https://github.com/witnesschain-com/dcl-operator-cli.git
```

- Build the binaries
```bash
cd operator-cli
make build
```

- test binaries
```bash
make test
```

- install
```bash
make install
```

This will install the `witnesschain-cli` in the `$HOME/.local/bin`.
